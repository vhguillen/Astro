Design Document

Purpose and goal of this website

The purpose of this website is to provide information easy to understand and follow to enthusiasts and beginners in the hobby of Astrophotography. It's a how-to website with basic information about amateur astrophography.

Audience and background

Enthusiasts and beginners in the hobby of Astrophotography. The recommended background of the audience is to have basic knowledge of astronomy and basic use of consumer cameras.

Scope of this website

To provide basic information on how to start in the hobby of Astrophotography. It contains some technical easy to follow information, however, deep technical information is not covered at this moment.

Navigational tools

This website provides navigational functionality based on regular links in a consistent way throughout the website.

Methodology - how to put site together

  • As a principle, I follow current design trends
  • I draw a skeleton of how I plan a page should look like
  • I write an HTML5 esqueleton and copy it to the rest of the files
  • Fill the content
  • Style the website
  • Write the scripts (or server programming if it's for work)

Resources I used
